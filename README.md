# Portfolio Template

This is a clean and effective portfolio template you can use to demonstrate your projects, skills, and experience.

You can view a live demo at <a href="https://mwestapps.com/">mwestapps.com</a>.

You can setup and customize your portfolio by the following these instructions:

**Install the necessary packages:**
```
npm install
```

**Start the project in development mode:**
```
npm run dev
```

**Enter your portfolio details:**

Edit `src/portfolio/details.ts` with your portfolio details and add any supporting images to `src/public/assets/images/`.

Added your name under the `author` field in `package.json`.

**Build the and deploy your portfolio:**
```
npm run generate
```

Once complete, upload the entire `.output/public/` directory to your server.
